# 🎧 DJ Josias - Site Oficial

Bem-vindo ao repositório do **Site Oficial do DJ Josias**.

Este projeto foi desenvolvido para apresentar os serviços do DJ de forma moderna, responsiva e profissional, permitindo que clientes conheçam seu trabalho e solicitem orçamentos facilmente.

---

## 🚀 Funcionalidades

- 🏠 Página inicial moderna
- 🎵 Apresentação do DJ
- 📖 Seção "Sobre"
- 🎧 Lista de serviços
- 🖼️ Galeria de fotos
- 📱 Botão para contato via WhatsApp
- 📍 Seção de contato
- 📄 Rodapé personalizado
- ✨ Animações suaves
- 📱 Layout responsivo para celulares, tablets e computadores
